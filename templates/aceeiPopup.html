ACEEI

Approximate Competitive Equilibrium from Equal Incomes

The goal of this algorithm is to determine a fair distribution of courses

for students that meets the Envy-Free constraints (EF-TB).

Iteration Process:

  • Optimal Budget Perturbation:

    In each iteration, the algorithm searches for the best adjustment in students' budgets based on the current course prices. This adjustment is determined using a linear programming approach.

  • Market Clearing Error:

    The algorithm then measures how well this budget adjustment clears the market, i.e., how closely the supply and demand match.

  • Price Update:

    Based on the market clearing error, the algorithm updates the course prices.

Termination:

The algorithm iterates until the market clearing error is minimized to zero, indicating that supply perfectly matches demand.

Linear Programming:

The linear program aims to minimize the market clearing error while meeting the following constraints:

  • Positive Prices: The demand for each course should match its capacity plus any excess demand.
  • Zero Prices: The demand for a course should be at least equal to its capacity plus any excess demand.

Fairness Constraint:

  • Each student should receive exactly one set of courses.
  • To ensure fairness and prevent envy, additional constraints are applied. If a student with a higher initial budget receives a specific course bundle, no student with a lower budget should receive the same bundle. This prevents jealousy among students.

Pseudocode Overview:

  • Initialize: Set initial prices and budgets.
  • Repeat: Until the market clearing error is zero:
    • Compute the optimal budget adjustment using linear programming.
    • Calculate the market clearing error for this adjustment.
    • Update the prices based on the calculated error.
  • Ensure: Each student receives one bundle of courses and enforce fairness constraints to avoid envy.
