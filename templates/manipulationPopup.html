Find Manipulation Algorithm

The goal of this algorithm is to determine if a course distribution algorithm is strategy-proof.

An algorithm is strategy-proof if students cannot gain any advantage by lying about their true preferences.

This ensures fairness and integrity in the course allocation process.

Parameters:

  • Mechanism: The method used for allocating courses.
  • True Utility: The actual utility of the student.
  • Criteria for Profitable Manipulation:
    • Randomness: Sample only the initial budgets.
    • Population: Sample the initial budgets and the utilities of other students.
    • Eta: A parameter used to adjust the student's preferences.

Algorithm Steps:

  1. Define v0 as the student's original utility from the input.
  2. Create a list of false utilities:
    • For each course, create two utility vectors:
      • In the first vector, the value of the current course is set to utility * eta.
      • In the second vector, the value is set to utility / eta.
  3. Find v*:
    • Choose a criterion for manipulation and run the mechanism multiple times for each false report and the true report.
    • Calculate the expected utility for each bundle from the reports.
    • Select the report that yields the maximum utility, which is v*.
  4. If v* is the same as the original utility v0, then the algorithm finishes.
  5. Otherwise, update v0 to the new utility v* and continue to find a better manipulation.
