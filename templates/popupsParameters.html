The algorithm is designed to find budgets and prices that make the allocation fair and envy-free. To achieve this, the parameter epsilon (ε) is used. Epsilon defines a range around the student's initial budget within which the algorithm checks for fairness. For example, if epsilon is set to 0.5 and a student's initial budget is 5, then the algorithm will consider budgets ranging from 4.5 to 6.5 for this particular student.
